# [Coursera] Unsafe Code in .NET: Boosting Performance in a Trading Application

This repository contains code from the Coursera course "Unsafe Code in .NET: Boosting Performance in a Trading Application". The course covers low-level .NET concepts like unsafe code, unmanaged memory, pointers, and value types (which are stack-allocated and faster than heap-allocated reference types). The example project is a basic order book for a trading app.

Check the [Key_Takeaways_Document](./assets/Key_Takeaways_Document.pdf) PDF document for more details.
